<p>
Hello and welcome from the team at Affiliate Links Expert , the ultimate Free Wordpress plugin for affiliate marketing.
</p>

<p>
As a website owner, you know the importance of having a successful affiliate marketing program. With <strong>Affiliate Links Expert</strong>, you can easily manage all your affiliate data from the admin panel. You can see how many pages were viewed via each affiliate link, how much money each user earned, and you can remove, restore or block users and affiliate links as necessary.
</p>

<p>
But <strong>Affiliate Links Expert</strong> is not just for website owners. It's also for users who want to create affiliate links and share them through social media platforms. With a user account, they can create affiliate links and see all the links they have created. They can also see how much money they earned and how much money the website owner paid them.
</p>

<p>
You can use <strong>Affiliate Links Expert</strong> without WooCommerce plugin to create affiliate links and tracking data about visits. However, you cannot track the number of items purchased without the WooCommerce plugin. To track the number of items purchased, use the WooCommerce plugin or integrate your payment methods with the <strong>Affiliate Links Expert</strong> plugin. For help with the integration, contact the creator of the plugin <a href="https://markomaksym.com.ua/" rel="ugc">Maksym Marko</a> or use your own developer.
</p>

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Affiliate Links Expert'
3. Activate the plugin from your Plugins page.
4. Create a page for partners’ cabinets on your website and place this shortcode [mxalfwp_partner_cabinet] there.

= From WordPress.org =

1. Download 'Affiliate Links Expert'.
2. Upload the 'Affiliate Links Expert' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'Affiliate Links Expert' from your Plugins page.
4. Create a page for partners’ cabinets on your website and place this shortcode [mxalfwp_partner_cabinet] there.
